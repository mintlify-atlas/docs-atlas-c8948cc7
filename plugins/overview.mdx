## Plugin Management

nvim-luffy uses [Lazy.nvim](https://github.com/folke/lazy.nvim) as its plugin manager. Lazy.nvim is a modern, fast, and feature-rich plugin manager that provides:

- Automatic lazy-loading of plugins
- Easy plugin configuration
- Built-in update checker
- Performance optimization
- Clean plugin specifications

<Note>
  All plugins are configured as Lua modules in the `lua/leyland/plugins/` directory for easy organization and maintenance.
</Note>

## Plugin Organization

Plugins are organized into two main import paths:

```lua
{ import = "leyland.plugins" }       -- Core plugins
{ import = "leyland.plugins.lsp" }   -- LSP-specific plugins
```

Each plugin is defined in its own Lua file, making it easy to:
- Enable/disable plugins by removing/adding files
- Customize individual plugin configurations
- Keep the codebase organized and maintainable

<Tip>
  To add a new plugin, create a new `.lua` file in `lua/leyland/plugins/` that returns a Lazy.nvim plugin specification.
</Tip>
