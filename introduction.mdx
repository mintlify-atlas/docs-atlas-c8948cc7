**nvim-luffy** is a modern, feature-rich Neovim configuration built for developers who want a powerful IDE experience right out of the box. Inspired by the adventurous spirit of One Piece, this configuration helps you navigate your codebase with the determination of a pirate king.

<Note>
  "Maybe the One Piece was the commits we made along the way."
</Note>

## Why nvim-luffy?

nvim-luffy combines the best plugins and configurations from the Neovim ecosystem into a cohesive, well-tuned setup that just works. No more hours spent configuring LSP servers, fuzzy finders, or color schemes - everything is ready to go.

### What's Included

<AccordionGroup>
  <Accordion title="Core Editor Features">
    - **Smart defaults**: Relative line numbers, system clipboard integration, and intelligent search
    - **Treesitter**: Advanced syntax highlighting and code understanding for 29+ languages
    - **Auto-completion**: Intelligent suggestions with nvim-cmp, LuaSnip snippets, and LSP integration
    - **Auto-pairs**: Automatic bracket and quote pairing
  </Accordion>

  <Accordion title="Development Tools">
    - **LSP Servers**: Pre-configured for TypeScript, JavaScript, HTML, CSS, Tailwind, Svelte, Lua, GraphQL, Python, and more
    - **Formatters**: Prettier, Stylua, Black, and isort ready to go
    - **Linters**: ESLint and Pylint integration
    - **Git Integration**: Neogit and Diffview for powerful version control
  </Accordion>

  <Accordion title="Navigation & Search">
    - **Telescope**: Fuzzy file finder, live grep, recent files, and keybinding search
    - **Harpoon 2**: Mark and instantly jump between your most important files
    - **Flash.nvim**: Lightning-fast cursor movement with search-based navigation
    - **Todo Comments**: Find and highlight TODO, FIX, NOTE, and other annotations
  </Accordion>

  <Accordion title="UI & Appearance">
    - **Multiple themes**: 6 carefully selected colorschemes (default: Bamboo)
    - **Alpha dashboard**: Custom One Piece-themed ASCII art welcome screen
    - **Minimal statusline**: Clean winbar showing file info without clutter
    - **Trouble**: Beautiful diagnostics and quickfix list
  </Accordion>
</AccordionGroup>

## Philosophy

nvim-luffy follows these guiding principles:

1. **Opinionated but flexible**: Sensible defaults that work immediately, with easy customization when you need it
2. **Modern tooling**: Uses the latest Neovim features and well-maintained plugins
3. **Performance first**: Lazy-loading and optimized configurations ensure fast startup times
4. **Developer experience**: Every plugin and keybinding is chosen to enhance your workflow

<Warning>
  This configuration requires Neovim 0.9+ and uses Space as the leader key. Make sure to check the [prerequisites](/installation#prerequisites) before installing.
</Warning>
